foundation-template
===================

Foundation 5 project template 

[Base - generator zf-5](https://github.com/juliancwirko/generator-zf5)

## Quickstart
Watch runs on 127.0.0.1:9000
```
npm install
bower install
grunt build
grunt
```

Dist runs on 127.0.0.1:9001
```
npm install
bower install
grunt build
grunt publish
grunt server-dist
```

### Grunt
* sass
* jshint
* connect
* usemin
* watch
* clean
* concat
* mincss
